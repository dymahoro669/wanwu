## 一、产品定位

**富泽经纪资产交易中心** 是一款基于"万物皆资产"理念的企业经营管理系统。

通过移动端与后期管理后台的协同，实现：

- **资产数字化**：将企业知识、工具、流程等转化为可交易的数字资产
- **价值透明化**：通过账户体系让每一份贡献都被看见、被记录、被回报
- **激励游戏化**：通过等级系统、成就体系提升用户参与度

**核心理念：**

- 万物皆资产

**核心设计原则：**

1. **温暖优先** — 结算不是惩罚，是价值流转的证明
2. **可见即动力** — 贡献被看见，激励自然产生
3. **互动创造价值** — 借用、感谢、协作让资产升值
4. **够用就好** — 不让系统成为负担

## 三、核心业务流程

### 3.1 租用完整流程

```
[发起租用] → [查看日历选择时段] → [填写目的] → [提交请求]
                                          │
                          ┌───────────────┴───────────────┐
                          ▼                               ▼
                   [需要审批]                      [跳过审批]
                          │                               │
                          ▼                               ▼
                   [等待审批]                        [租用开始]
                          │                               │
              ┌───────────┴───────────┐                   │
              ▼                       ▼                   │
         [审批拒绝]              [审批通过]                   │
              │                       │                   │
              ▼                       ▼                   │
           [借用失败]            [日历同步展示]                │
                                      │                   │
                                      ▼                   │
                                 [使用资产]                │
                                      │                   │
                                      ▼                   │
                                 [租用结束]                │
                                      │                   │
                            ┌─────────┴─────────┐          │
                            ▼                   ▼          │
                      [按时归还]           [延期/损坏]        │
                            │                   │          │
                            ▼                   ▼          │
                      [感谢信可选]         [额外金额消耗]     │
                            │                   │          │
                            └─────────┴─────────┘          │
                                      │                   │
                                      ▼                   │
                             [金额结算]                    │
                                      │                   │
                                      ▼                   │
                             [单向评价]                    │
                                      │                   │
                                      ▼                   │
                             [记录存档]                    │
```

**时间精度：** 最小精确到30分钟

**归还确认机制：**

- 租用方自助申报归还：租用方在"我的租用"中点击"确认归还"
- 资产拥有者确认：拥有者收到归还通知后进行确认
- 如超过预计归还时间3天未确认，系统自动标记为已归还并触发结算

**结算规则：**

- 租用时冻结金额
- 按时归还：金额从冻结转为实际消耗
- 提前归还：剩余金额返还，计算实际消耗
- 延期归还：产生额外金额消耗
- 损坏丢失：按资产价格赔偿

**结算权重说明：**

- 高等级用户获得的额外结算权重（+5%/+10%/+15%）由平台补贴，不影响租用方支付金额
- 结算时：资产拥有者获得 = 基础金额 × (1 + 等级权重补贴)
- 示例：100元租用，6级用户获得105元（其中5元由平台额外结算）

### 4.5 评价功能

**评价触发时机：**

- 租用完成 → 仅租用方评价资产拥有者

**评价维度：**

| 维度    | 说明           | 分值   |
| ----- | ------------ | ---- |
| 整体满意度 | 本次交易整体评价     | 1-5星 |
| 资产质量  | 描述与实际符合程度    | 1-5星 |
| 响应速度  | 资产提供者的响应及时程度 | 1-5星 |
| 专业程度  | 资产提供者的专业能力   | 1-5星 |

**评价规则：**

- 完成交易后48小时内必须完成评价
- 未评价则自动默认好评
- 差评后被评价方可申诉
- 评价提交后24小时内可修改
- 可选择匿名评价

**评价在移动端的位置：**

| 位置    | 功能           | 说明               |
| ----- | ------------ | ---------------- |
| 我的租用  | **待评价** 标签页  | 进行中租用，待评价时显示红点提醒 |
| 我的租用  | **评价记录** 标签页 | 查看历史评价           |
| 资产详情页 | 评价汇总展示       | 整体评分、评价数量、最近评价   |
| 个人主页  | 收到的评价汇总      | 作为资产拥有者的评价统计     |

## 十一、交互设计

### 11.1 全局交互原则

1. **温暖反馈** - 每个操作都应有明确的正向反馈
2. **渐进披露** - 不一次性展示所有功能，通过引导逐步展示
3. **容错设计** - 误操作可撤销，重要操作有确认
4. **即时响应** - 本地操作即时反馈，远程操作显示加载中状态

---

### 11.2 关键交互状态

| 场景   | 状态设计                 |
| ---- | -------------------- |
| 加载中  | 显示骨架屏或加载中动画，避免空白 |
| 操作成功 | 轻提示提示 + 数据即时更新     |
| 操作失败 | 明确错误原因，提供重试入口        |
| 网络异常 | 离线缓存 + 网络恢复后自动同步     |
| 空状态  | 友好的空状态提示 + 引导操作      |

---

### 11.3 推送通知策略

| 通知类型 | 触发时机       | 优先级 |
| ---- | ---------- | --- |
| 租用审批 | 有人申请租用您的资产 | 高   |
| 租用结果 | 您的租用申请被审批  | 高   |
| 感谢信  | 收到感谢信      | 中   |
| 成就达成 | 完成一个成就     | 中   |
| 系统通知 | 重要系统消息     | 高   |

### 13.21 关键业务规则

| 维度 | 规则 | 说明 |
|---|---|---|
| 租用时段 | 最小 30 分钟 | 超过 30 分钟按实际使用时长结算 |
| 并发竞争 | 先提交者先得 | 同一时段以提交时间戳为准 |
| 超期结算 | 72 小时强制 | 超期 72 小时自动结算并冻结账户 |
| 感谢信免费 | 每租用1次1封 | 每次租用可发送一封免费感谢信 |
| 感谢信有偿 | 100 元/封 | 每月 100 元额度，不可跨月累积 |
| 感谢信间隔 | 6 个月冷却 | 有偿感谢同一人需间隔 6 个月以上 |
| 结算权重 | 等级加成 | Lv.5 以上每级 + 5% 结算收益 |
| 提现周期 | 每月 1-15 日 | 其他时间入口灰度锁定 |
| 提现最低 | 100 元 | 低于 100 元无法发起提现申请 |
| 冻结机制 | 租用预冻 | 提交租用时冻结对应金额 |
| 归还确认 | 三重机制 | 租用方申报 → 拥有者确认 → 超期自动标记 |
| 审批流程 | 可选配置 | 资产创建者可开启/关闭审批 |
| 审批超时 | 24 小时 | 超时自动通过，通知双方 |
| 归还提醒 | 智能定时 | 根据租用时长自动计算最优提醒时机 |
