## 1. 产品概述

### 1.1 产品定位

万物资产交易中心是一个企业级资源共享与交易平台，致力于通过数字化手段优化企业内部资源的流通与利用效率，实现"万物皆可共享，万物皆有价值"的核心理念。

### 1.2 目标用户

- 企业内部员工
- 部门资产管理员
- 行政管理人员
- 资源共享需求者

### 1.3 核心功能

- **资产浏览与搜索**: 支持按分类、标签、创建者等多种维度浏览资产
- **资产交易**: 支持租用、领取、预约等多种交易方式
- **数据中心**: 展示公司级资产活跃数据、排行榜、成就系统
- **个人中心**: 用户资产管理和交易记录
- **安全认证**: 支持指纹识别快捷登录

### 2.7 设计原则

1. **温暖亲和**: 使用暖色调（橙色、棕色系）营造温暖、专业的氛围
2. **清晰层次**: 通过容器层级和阴影深度区分内容层次
3. **一致体验**: 统一的间距、圆角、阴影规范确保体验一致性
4. **微妙动效**: 轻量的悬停、点击反馈提升交互感受

#### Banner轮播区域

- **高度**: 55vh，最小450px
- **轮播方式**: 水平滚动，snap-x snap-mandatory
- **图片比例**: Slide 1 使用 16:9，Slide 2 使用其他比例
- **渐变遮罩**: `from-black/50 via-black/20 to-background`
- **内容位置**: 底部，距底部100px，左右各24px

**Banner标签样式**:
- 全新发布: `bg-primary text-white`
- 限时特惠: `bg-tertiary text-white`
- 字体: Label SM, 10px
- 圆角: `rounded-md`
- 内边距: `px-2 py-1`

#### 搜索栏

- **位置**: Banner下方，距底部6px
- **容器样式**: `rounded-2xl`，阴影 `0 8px 32px rgba(0,0,0,0.15)`
- **背景**: `bg-white/25`，毛玻璃效果 `backdrop-blur-xl`
- **边框**: `border-white/40`
- **搜索图标**: 白色，左侧，距左边12px
- **输入框**: 透明背景，白色文字，占位符 `rgba(255,255,255,0.5)`
- **搜索按钮**: `bg-primary`，白色文字，距右侧8px

#### 分类横向滚动

- **布局**: Flexbox，`overflow-x-auto`
- **胶囊按钮**:
  - 全部（选中）: `bg-primary-container text-on-primary`
  - 其他: `bg-surface-container text-on-surface-variant`
- **间距**: `gap-3`
- **内边距**: `px-5 py-2.5`
- **圆角**: `rounded-full`

#### 资产卡片

**精选推荐卡片（全宽）**:
- **容器圆角**: `rounded-[24px]`
- **图片比例**: `aspect-[16/9]`（移动端）, `aspect-[21/9]`（桌面端）
- **标签位置**: 左上角，`top-4 left-4`
- **内容布局**: 移动端垂直，桌面端水平
- **阴影**: 默认 `0 8px 24px rgba(109,91,70,0.06)`，悬停 `0 12px 32px rgba(109,91,70,0.12)`

**普通资产卡片**:
- **列数**: 移动端2列，桌面端3-4列
- **布局**: 瀑布流 (columns)
- **间距**: `gap-gutter` (16px)
- **图片比例**: 资产类 `16:9`，文档类 `3:4`
- **图标卡片**: 渐变背景 + 大图标（如技能类）

**卡片内容**:
- 标题: `font-headline-md`，最多2行
- 评分: 星号图标 + 数字
- 创建者: 头像 + 名字 + 部门
- 价格: 大号主色字体
- 标签: 小号青绿色背景
- 按钮: 圆角胶囊按钮

### C. 状态定义

| 状态 | 颜色 | 说明 |
|------|------|------|
| 默认 | - | 组件标准状态 |
| 悬停 | 略深 | 鼠标悬停效果 |
| 激活 | 略暗 | 点击按下效果 |
| 禁用 | 灰色 | 不可操作状态 |
| 加载中 | 骨架屏 | 内容加载中 |

### D. 动效规范

| 动效类型 | 时长 | 缓动函数 |
|----------|------|----------|
| 悬停效果 | 200-300ms | ease-out |
| 点击反馈 | 150ms | ease-in |
| 页面过渡 | 300ms | ease-in-out |
| 脉冲动画 | 1500ms | ease-in-out (循环) |
